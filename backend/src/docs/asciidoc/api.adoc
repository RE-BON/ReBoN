== Common

=== Exception

|===
| 상태 코드 | 설명

| 400 | `잘못된 요청`
| 401 | `권한 없음`
| 500 | `서버 에러`
|===

```json
  "message": "에러 상세 내용"

```

=== Pagination & Sorting
|===
| 값 | 타입 | 필수 | 설명

| page | Integer | false | 원하는 페이지
| size | Integer | false | 한 페이지에 보여줄 데이터 수
| sort | String | false | 정렬 기준
|===
```
ex) 1페이지 데이터 10개 생성일 오름차순 정렬
https://localhost:8080/api/shops?page=0&size=10&sort=createdAt,asc
```
